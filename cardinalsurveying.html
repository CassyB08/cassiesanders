Cardinal Surveying

Web Redesign

Cardinal Surveying is a family-owned, award-winning, professional land surveying agency located in Missouri. This company has a passion for educating realtors about land surveying.

Website Redesign

A customer of Shortcut Solutions, the project goal for Cardinal Surveying was a completely new, fully responsive website that serves all of the company’s needs. From an outdated website to a Joomla application, Cardinal Surveying performs better in search engines, provides website visitors with the information they need, and helps to convert prospective students! In my quest to accomplish those goals, I ensured the website was Search Engine Optimized (SEO), easy to navigate, and showcased their informational blog articles.

Project Scope

  • Custom, Responsive Website Design
  • Search Engine Optimization
  • Graphic Design
  • Website Content Management System (CMS)
