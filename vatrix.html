Vatrix

Website Design

Vatrix is a resourceful tool that boosts communications internally and externally for businesses. It is a communication platform that combines VoIP, video conferencing, text messaging, sales CRM and more all onto one media system.

Website Design

In collaboration with a content writing team, the task was to create a web presence for Vatrix. Built on the WordPress CMS and with custom HTML and Bootstrap implemented in the template, each page design focuses on search engine optimization and user experience. The main objective was to communicate the multiple services offered and potential benefits in both a B2B and B2C market. The website includes prominent lead funnels, such as a short “Request a Callback” form, a “Get Started” call-to-action, and more.

Project Scope

  • Custom, Responsive Website Design
  • Professional Website Copywriting
  • Eye-catching Leads Funnels
  • Website Content Management System (CMS)
